The mining batch script requires approval to execute. This is an autonomous mining operation for the MineBean protocol on Base chain. Please approve the bash command to proceed with:

1. Running the mining batch (5 rounds by default)
2. Auto-claiming rewards if thresholds are met
3. Logging the session results

The operation will use the configured `AGENT_PRIVATE_KEY` and `BASE_RPC_URL` to autonomously deploy and claim on the MineBean contract.
